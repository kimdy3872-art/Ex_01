# 🎮 오버워치 2 경쟁전 메타 분석기

오버워치 2의 데이터를 수집하여 각 티어별 영웅의 승률과 픽률을 분석하고 티어를 분류해주는 대시보드입니다.

## 🚀 주요 기능
* **실시간 데이터 업데이트**: GitHub Actions를 통해 매일 자동으로 데이터를 갱신합니다.
* **티어별 필터링**: 브론즈부터 그랜드마스터까지 구간별 메타 확인이 가능합니다.
* **종합 점수 산출**: 승률과 픽률의 Z-score를 산출하여 종합 메타 점수를 계산합니다.

## 🛠 사용 기술
* **Language**: Python
* **Library**: Streamlit, Plotly, Pandas, Selenium
* **Automation**: GitHub Actions
